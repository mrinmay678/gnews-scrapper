How to use this app
```

Install python and then node in your machine

First create a Virtualenv

For Linux or Mac Users:

virtualenv env 

For All:

python -m venv env

then pip install -r newscrapper/requirements.py to install python dependencies

For Continuous updation of database run the continuousUpdation.py file

as 
python continuousUpdation.py

newsscrapper is the django project 

frontend is the react project

To run django project

run python newscapper/manage.py runserver

then move to frontend folder and do npm install to install node dependencies

To run react project

npm start

if you have yarn

then yarn start
